HTMLTableKit ES6 Module Demo

⚠️ Important: ES6 Module Usage

This demo uses ES6 modules. To run it, you need to serve it through a web server, not open it directly as a file.

Quick start options:

  • Python: python -m http.server 8000
  • Node.js: npx http-server
  • VS Code: Use the "Live Server" extension
